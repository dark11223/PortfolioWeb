Projects

Youtube Clone

This is my first project, which I created out of curiosity about how YouTube is built—especially its layout. I decided to recreate it using only HTML and CSS. It’s a simple and static version, and I only managed to make the homepage, but it still satisfied my curiosity and helped me become more proficient in using HTML and CSS.

Rock Paper Scissor

This is a simple project I created—a Rock, Paper, Scissors game with a retro classic theme. It features auto-play functionality, sound effects for a more engaging experience, and a game statistics tracker that records wins, losses, and draws. The design and layout were inspired by old-school arcade aesthetics, making the interface both fun and nostalgic.

Todo List

A simple yet functional Todo List project that allows users to add, manage, and track their daily tasks efficiently. It includes features such as task sorting, filtering by status or priority, and setting priority levels to help organize tasks based on importance. The interface also supports a dark and light mode toggle for better user experience and accessibility.
